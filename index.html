HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • If you open an element or tag make sure you close it
  • The tag ul creates an unorderd list of items.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways in which you can add css to your code inline, internal and external
  • When using the external CSS method you must link the stylesheet to the HTML using href attribute

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git push: Lets us upload local repository content to a remote repository
  • git pull: Allows us to fetch and download content from a remote repository and immediately update the local repository to match that content

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript file is read from top to bottom. This is known as control flow
  • Math.floor() and Math.random() are both properties of JavaScript's built-in Math object that can be used together to generate a random number
